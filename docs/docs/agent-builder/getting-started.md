# Getting Started

The Agent Builder lets you create custom AI agents by combining skills and instructions together. Think of it like assembling a specialist — you define what it knows (instructions), what it can do (skills), and how it behaves (model parameters).

## Core Concepts

An agent is made up of three building blocks:

- **Instructions** — Guidelines that shape how the agent responds. These set the tone, rules, and priorities for the agent's behavior.
- **Skills** — Specific capabilities or areas of expertise. Skills define what the agent is good at and when those abilities should activate.
- **Agent** — The final composition. An agent combines a model with your chosen skills and instructions into something you can interact with.

The relationship is simple: **Instructions + Skills = Agent**.

## Creating Your First Agent

1. Navigate to the **Agent Builder** from your sidebar.
2. You will land on the **Discover** page where you can browse existing agents, skills, and instructions.
3. Click **Create** to open the builder.
4. You will see three tabs at the top: **Agents**, **Skills**, and **Instructions**.

We recommend starting by creating a few instructions and skills first, then combining them into an agent. However, you can create them in any order.

### Quick Start

If you want to jump right in:

1. Go to the **Create** page.
2. Click the **New** button and select **Agent**.
3. Give it a name and description.
4. Configure the model parameters (or leave the defaults).
5. Optionally attach any existing skills and instructions.
6. Click **Create**.

That's it — you now have a working agent you can test right from the builder.

For more details on each building block, see the dedicated pages for [Agents](./agents), [Skills](./skills), and [Instructions](./instructions).
