# Adding References

One of the powerful parts about Clove is investigating, referencing, and summarizing documents. To upload documents in a new chat or an existing chat is the same, just click the upload arrow on the left hand corner of the input.

This will let you uplaod a variety of documents from word, power point, pdf (both ocr and non-ocr), excel code files, markdown, text, and more.

Note, you may be alerted to the size of your files. Under the hood, the actual size of the file does not matter: the amount of content in the file is what matters. If you get alerts about files being too large, you can either use part of the file and not the entire file.

Or you can use a collection (which just so happens to be the next section).
