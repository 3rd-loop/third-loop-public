# Getting Started

Clove is an interface for working with the next generation of natural language processing models (https://www.geeksforgeeks.org/nlp/natural-language-processing-overview/).

Clove has a ton of functionality but you can get started by simply hitting the "New Chat" button in your sidebar and asking a question.

We host Clove in a secure environment for both you and your company. Because of this, you can upload information that would typically be prohibative for public AI looks like Chat GPT.
