# Getting Started

Clove is an interface for working with the next generation of natural language processing models. It is hosted in a secure environment for both you and your company, which means you can upload information that would typically be off-limits for public AI tools like ChatGPT.

## Your First Chat

You can get started by clicking the **New Chat** button in your sidebar and asking a question. Clove will respond in real-time, streaming its answer as it generates it.

From there you can continue the conversation, upload files for Clove to reference, or switch to a different agent or model using the dropdown near the input field.

## Navigating Clove

The sidebar on the left is your main hub. From there you can:

- **New Chat** — Start a fresh conversation.
- **Collections** — Manage your uploaded files and documents for knowledge retrieval.
- **Workflows** — Create and manage repeatable processes.
- **All Chats** — Browse and search through your conversation history.
- **Settings** — Customize your experience (theme, instructions, usage, templates).

Your recent chats also appear in the sidebar for quick access. Hover over any chat to rename it, view details, or delete it.

## Agents

Clove supports multiple AI agents and models. You can select which agent to use from the dropdown in the chat input area. Different agents may have different capabilities, tone, or expertise depending on how they were configured in the Agent Builder.

You can switch agents mid-conversation if needed.

## What Can Clove Do?

- Answer questions and have conversations
- Read, analyze, and summarize documents you upload
- Search your collections for relevant information (RAG)
- Generate and render code, tables, and diagrams (including Mermaid)
- Walk you through structured workflows for repeatable processes
- Follow custom instructions you set in your settings
